Exercício 1 - Variável undefined

  • Para declarar undefined - Declaração da variável sem atribuir um valor à ela, exemplo: tipo variável nome;
  • Para declarar vazio(void) - Declaração da variável com atribuição de nenhum char, exemplo: tipo variável nome = "";
  • Para declarar com valor - Declaração com o operador de atribuição = e o tipo de dado (int, float ...), exemplo: tipo variável nome = dado;
  • Saídas no console!

Exercício 2 - Como usar operadores relacionais

  • Operador != - Operador binário Not Equal, retorna Verdadeiro quando os elementos comparados são diferentes.
  • Operador == - Operador binário Equal to, retorna Verdadeiro quando os elementos tem valores iguais, não compara o tipo.
  • operador >= - Operador binário Greater than or equal to, maior que ou igual, retorna Verdadeiro quando o primeiro elemento (esquerda) é maior que o segundo (direita depois do operando) ou quando são iguais.
  • Saídas no console!
